Case Study / 3D Orbital

Project 01

System Dynamics.

Tools

Software Stack

Pipeline used Blender for scene building, keyframing, and rendering, with lightweight post-editing.

Design

Visual Objective

Emphasizes smooth orbital paths, clear focal points, and readable scale relationships between bodies.

Build

Key Features

Final output includes animated camera travel, lighting, and an interactive data calculator below.

Development

Production Basics

  • ● Orbital paths were blocked first to lock scene rhythm before camera passes.
  • ● Lighting was tuned for silhouette clarity so each planet remains identifiable.
  • ● Render output was exported at presentation-friendly resolution.
